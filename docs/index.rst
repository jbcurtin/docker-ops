#################
Docker Operations
#################

A utility that'll package up Docker Images and push them around the internet

Scanner
#######

.. code-block:: bash

    $ pip install docker-ops -U
    $ IMAGE_REGISTRY_DOMAIN='hub.docker.com' docker-ops.py -s -d images


Planned Support
---------------

* Nexus
* Gitlab
* Docker Hub
